# 小程序接入流程

TIP

小程序目前接入方式是通过提供示例代码的方式，开发者可以通过示例了解如何在项目中如何使用数字人，也可将示例中的代码直接拷贝至项目中即可使用数字人

# 如何导入

TIP

若存在同名目录，则将目录内的文件复制至对应目录即可

SDK将会以dist目录的方式提供，关注以下目录：

  • common
  • Components
  • pages
  • static

1.将common、components、static目录复制至您项目的miniprogram目录下

2.将pages下的 index、cmcc 复制至您项目的pages目录下

# 如何配置

TIP

文件路径及文件名称可根据实际情况调整

在app.json的pages新增页面：

"pages": [
    "pages/index/index", // 入口页
    "pages/cmcc/cmcc", // 播放页 
 ]

在app.js中新增：

require('./common/runtime.js')
require('./common/vendor.js')
require('./common/main.js')

跳转页面 ../index/index" 即可开始测试
