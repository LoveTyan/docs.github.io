# 小程序接入流程

小程序目前接入方式是通过代码导入的方式接入

# 如何导入

SDK将会以dist目录的方式提供，关注以下目录：

  • common
  • Components
  • pages
  • static

1.将common、components、static目录复制至您项目的miniprogram目录下

2.将pages下的 entry、live 、vod 复制至您项目的pages目录下

若存在同名目录，则将目录内的文件复制至对应目录即可

# 如何配置

文件路径请根据实际情况调整

在app.json的pages新增页面：

[
    "pages/home/index", // 入口页
    "pages/live/live", // 直播
    "pages/vod/vod" // 点播
  ]

在app.js中新增（：

require('./common/runtime.js')
require('./common/vendor.js')
require('./common/main.js')

跳转页面 ../entry/index" 即可开始测试
